(B) CREATE
POST {{baseUrl}}/items
Body → raw → JSON
{
  "name": "apple",
  "price": 1234
}

Headers: Content-Type: application/json

Pre-request Script(랜덤 데이터로 보내기)

const rnd = Math.floor(Math.random() * 9000) + 1000;
pm.variables.set("rndName", "item_" + Date.now());
pm.variables.set("rndPrice", rnd);

(Body를 변수로 바꾸면 매번 자동 생성):
{
  "name": "{{rndName}}",
  "price": {{rndPrice}}
}

(Tests, 201+itemId 저장)
pm.test("201 Created", () => pm.response.to.have.status(201));

const item = pm.response.json();
pm.test("has id/name/price", () => {
  pm.expect(item).to.have.property("id");
  pm.expect(item).to.have.property("name");
  pm.expect(item).to.have.property("price");
});

pm.environment.set("itemId", item.id); // 다음 요청들이 이 id로 동작

(C) READ 단건(방금 만든 itemId로)

GET {{baseUrl}}/items/{{itemId}}

(Tests)
pm.test("200 OK", () => pm.response.to.have.status(200));
const item = pm.response.json();
pm.test("id matches env itemId", () => {
  pm.expect(item.id).to.eql(Number(pm.environment.get("itemId")));
});

(D) PUT 전체 수정(둘 다 보내야 함)

PUT {{baseUrl}}/items/{{itemId}}
Body:

{
  "name": "updated_full",
  "price": 7777
}

(Tests)

pm.test("200 OK", () => pm.response.to.have.status(200));
const item = pm.response.json();
pm.test("updated", () => {
  pm.expect(item.name).to.eql("updated_full");
  pm.expect(item.price).to.eql(7777);
});


(E) PATCH 부분 수정(하나만 보내도 됨)

PATCH {{baseUrl}}/items/{{itemId}}
Body 예시1(가격만):

{ "price": 8888 }

(Tests)
pm.test("200 OK", () => pm.response.to.have.status(200));
const item = pm.response.json();
pm.test("price updated", () => pm.expect(item.price).to.eql(8888));
